# Deep Sea

Subnautica-meets-terraria in 2D using Godot


## Mechanics

### Subnautica
 - travel depth limited by player progression
 - snap-fit & modular base building
 - energy generators (solar, bioreactor, nuclear, ionic)
 - built-in AI/story driven guide

### Terraria
 - projectile based combat
 - unload to nearby chests inventory mechanic
 - fishing
 - well defined biomes
 - multiple progression routes (ranged, melee, magic)
